# Plant Room Profile 
An interactive 3D plant shop built with three.js to showcase developer projects, skill, and personality. This repo will take you through a cozy plant store where each plant is a project you can explore. 

***STATUS:*** Under active development. (Early preview).

## Core Concept
- ***3D Room Gallery:*** A plant shop environment where each plant represents a project.
- ***Project Interactions:*** Click/Hover to reveal details, screenshots, links, and tech stacks.
- ***Skills as Decor:*** Tools and skills appear as labels, shelves, posters, or props in the space.
- ***Performance First:*** Optimized models, textures, and lighting for smooth interaction.

## Tech Stack
  - three.js
  - JavaScrpit
  - TypeScript
